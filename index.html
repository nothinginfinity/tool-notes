Voice Memos are a human UI.
Tool Notes is agent memory.

Apple Voice Memos records and transcripts for humans. It has no MCP surface, no structured metadata, no vector search, and no safe tool access for LLMs. Tool Notes is designed from the agent side out.

🔒 Trapped transcripts

Apple Voice Memos locks your transcript inside a human-only app. No agent can search or reason over it.

🧠 Agent-accessible memory

Every Tool Notes transcript is stored, chunked, embedded, and exposed through MCP tools. ChatGPT and Claude can find it.

⚙️ MCP-compatible by design

Tool Notes exposes its state and actions to agents through safe, scoped tools — not UI scraping or clipboard hacks.

The flow

From spoken thought to agent-accessible memory in seconds.

01
Record

Voice Memos, Shortcut, or native app.

02
Ingest

POST transcript + metadata to /api/notes/ingest.

03
Store

D1 database. Structured record. Indexed.

04
Embed

Chunks embedded into Vectorize for semantic search.

05
Search

LLMs access through MCP tools. Search, summarize, extract.
